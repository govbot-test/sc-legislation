A bill TO AMEND THE SOUTH CAROLINA CODE OF LAWS BY ADDING SECTIONS 4-25-80, 4-25-290, AND 5-25-950 ALL SO AS TO REQUIRE THE INSTALLATION OF FIRE SPRINKLER SYSTEMS IN NEWLY CONSTRUCTED MULTI-FAMILY DWELLINGS PERMITTED TO BE BUILT AFTER JULY 1, 2026; AND BY ADDING SECTION 23-9-165 SO AS TO REQUIRE THE STATE FIRE MARSHALL TO PROMULGATE REGULATIONS TO DESIGNATE APPROPRIATE FIRE SPRINKLER SYSTEMS FOR NEWLY CONSTRUCTED MULTI-FAMILY DWELLINGS. Be it enacted by the General Assembly of the State of South Carolina: SECTION 1.  Article 1, Chapter 25, Title 4 of the S.C. Code is amended by adding: Section 4-25-80. All newly constructed multi-family dwellings permitted to be built after July 1, 2026, must install fire sprinkler systems. SECTION 2.  Article 3, Chapter 25, Title 4 of the S.C. Code is amended by adding: Section 4-25-290.  All newly constructed multi-family dwellings permitted to be built after July 1, 2026, must install fire sprinkler systems. SECTION 3.  Article 7, Chapter 25, Title 5 of the S.C. Code is amended by adding: Section 5-25-950.  All newly constructed multi-family dwellings permitted to be built after July 1, 2026, must install fire sprinkler systems. SECTION 4.  Article 1, Chapter 9, Title 23 of the S.C. Code is amended by adding: Section 23-9-165.  The State Fire Marshall must promulgate regulations to specify the appropriate type of fire sprinkler system for installation in multi-family dwellings as required by this act. SECTION 5.  This act takes effect upon approval by the Governor. ----XX---- This web page was last updated on April 15, 2026 at 01:49 PM
